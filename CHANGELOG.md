# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-13

### Added
- Initial release of ThetaData Local Proxy.
- Support for stock history EOD requests.
- Queue-based architecture for sequential request processing.
- REST API endpoints for submission, status, and results.
- In-memory storage with automatic cleanup of stale requests.
- Comprehensive documentation and community guidelines.
